# Leccion 04 - Condicionales y Logica Proposicional

Esta leccion covers condicionales en Python (`if`, `elif`, `else`), operadores de comparacion y logica proposicional con tablas de verdad.

## Idea central

Los condicionales permiten tomar decisiones en el programa. La logica proposicional permite combinar condiciones. Ambas cosas van de la mano para construir programas que evaluen diferentes situaciones.
