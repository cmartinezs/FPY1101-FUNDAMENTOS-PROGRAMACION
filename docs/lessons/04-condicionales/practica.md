# Practica

## Ejercicios guiados

1. Pedir un numero y mostrar si es par o impar.
2. Pedir dos numeros y mostrar el menor.
3. Pedir una letra y verificar si es mayuscula.
4. Pedir un numero y verificar si esta entre 20 y 30.
5. Pedir una edad y mostrar la categoria: Nino (0-12), Adolescente (13-17), Adulto (18-64), Adulto mayor (65+).

## Ejercicios autonomos

1. Pedir un año y verificar si es divisible por 4.
2. Pedir un numero y mostrar si es divisible por 3 y por 5 al mismo tiempo.
3. Pedir dos numeros y verificar si el primero es mayor y es par.
4. Pedir un nombre y verificar si comienza con la letra "A".
5. Pedir un numero y verificar si es diferente de cero.
6. Pedir una opcion (1, 2 o 3) y mostrar el mensaje correspondiente (1=Iniciar, 2=Pausar, 3=Salir).
7. Pedir la edad y si es Ciudadano (s/n) y verificar si puede votar (Ciudadano Y mayor de 18).
8. Pedir un caracter y verificar si es una vocal o una consonante.
9. Pedir un numero y verificar si es negativo O impar.
10. Pedir el peso y la altura, calcular el IMC y clasificar: Bajo peso (<18.5), Normal (18.5-24.9), Sobrepeso (25-29.9), Obeso (30+).
11. Pedir tres numeros y mostrar el mayor.
12. Pedir un numero y verificar si termina en 0.
13. Pedir la hora (0-23) y mostrar el turno: Madrugada (0-5), Manana (6-11), Tarde (12-17), Noche (18-23).
14. Pedir un numero y verificar si es primo (divisible solo por 1 y por si mismo).
15. Pedir dos valores booleanos y mostrar el resultado de AND, OR y NOT para cada uno.

## Ejercicio integrador

Crear un programa que simule un sistema de login simple:

1. Pedir el nombre de usuario.
2. Pedir la contrasena.
3. Verificar las siguientes condiciones:
   - El usuario no debe estar vacio.
   - La contrasena debe tener al menos 6 caracteres.
   - Si ambas son correctas, mostrar "Login exitoso".
   - Si el usuario esta vacio, mostrar "El usuario no puede estar vacio".
   - Si la contrasena tiene menos de 6 caracteres, mostrar "La contrasena debe tener al menos 6 caracteres".
   - Si ambas son incorrectas, mostrar ambos errores.

Ejemplo de salida esperada:

```
Usuario: 
Contrasena: hola
Error: El usuario no puede estar vacio
Error: La contrasena debe tener al menos 6 caracteres
```
