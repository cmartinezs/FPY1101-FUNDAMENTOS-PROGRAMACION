# Practica

## Ejercicios guiados

1. Crear una funcion con type hints que retorne el cuadrado de un numero.
2. Crear una funcion con type hints que muestre si un numero es positivo.
3. Crear una funcion con type hints que calcule el area de un circulo.
4. Crear una funcion con type hints que retorne el mayor de dos numeros.
5. Crear una funcion con type hints que calcule el promedio de 3 notas.

## Ejercicios autonomos

1. Funcion con type hints que sume dos flotantes.
2. Funcion con type hints que verifique si un numero es par (retorna bool).
3. Funcion con type hints que calcule el descuento (recibe precio y porcentaje, retorna precio final).
4. Funcion con type hints que reciba un nombre y retorne un saludo.
5. Funcion con type hints que calcule el volumen de un cubo (lado^3).
6. Funcion con type hints que acepte entero o flotante y retorne el valor absoluto.
7. Funcion con type hints y valor por defecto para el IVA (19%).
8. Funcion con type hints que retorne True si un texto tiene mas de 5 caracteres.
9. Funcion con type hints que calcule el perimetro de un circulo (2 * pi * radio).
10. Funcion con type hints que convierta grados a radianes.
11. Funcion con type hints que retorne el tipo de triangulo segun sus lados.
12. Funcion con type hints que acepte parametros variables y retorne la suma.
13. Funcion con type hints que valide que una contrasena tenga al menos 6 caracteres.
14. Funcion con type hints que calcule el IMC (peso / altura^2) y retorne la clasificacion.
15. Funcion con type hints que procese una lista de numeros y retorne el mayor.

## Ejercicio integrador

Crear un programa con las siguientes funciones con type hints:

1. `obtener_opcion() -> int` - Muestra el menu y retorna la opcion (1-4).
2. `calcular_area_cuadrado(lado: float) -> float` - Calcula el area.
3. `calcular_area_circulo(radio: float) -> float` - Calcula el area.
4. `calcular_area_triangulo(base: float, altura: float) -> float` - Calcula el area.
5. `mostrar_resultado(figura: str, area: float) -> None` - Muestra el resultado.

El programa debe pedir una opcion y los datos necesarios, luego calcular y mostrar el resultado.
