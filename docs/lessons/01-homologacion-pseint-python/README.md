# Leccion 01 - Homologacion entre PSeInt y Python

Esta leccion introduce Python usando como puente los conceptos que el estudiante ya conoce en PSeInt.

## Idea central

PSeInt ayuda a pensar la logica. Python obliga a escribir esa logica con una sintaxis real, precisa y ejecutable.
