# Practica

## Ejercicios guiados

1. Pedir el apellido del usuario y mostrarlo en pantalla.
2. Pedir un numero y mostrar su doble.
3. Pedir dos numeros y mostrar la resta.
4. Pedir una edad y mostrar si la persona es mayor o menor de edad.
5. Pedir dos numeros y mostrar suma, multiplicacion y residuo.

## Ejercicios autonomos

1. Pedir el nombre y la edad de una persona y mostrar ambos datos en una sola salida.
2. Pedir tres numeros y mostrar su promedio.
3. Pedir una nota y mostrar si esta aprobada o reprobada usando `if`.
4. Pedir una opcion entre 1 y 3 y mostrar un mensaje distinto para cada caso.
5. Mostrar los numeros del 1 al 10 usando `for`.
6. Pedir una clave hasta que el usuario escriba `python123`.
7. Pedir dos numeros y mostrar cual es mayor usando operadores de comparacion.
8. Pedir edad y permiso, y mostrar si una persona puede entrar usando un operador logico.
9. Pedir usuario y mostrar si no esta vacio usando `not`.

## Ejercicio integrador

Crear un programa que:

1. pida el nombre de un estudiante
2. pida una opcion de menu
3. pida dos numeros y use operadores aritmeticos
4. use una condicion con operadores de comparacion o logicos
5. muestre una repeticion corta con `for`
