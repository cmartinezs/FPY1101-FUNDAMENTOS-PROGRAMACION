# Leccion 07 - Listas y Diccionarios

Esta leccion cubre las estructuras de datos fundamentales de Python: listas para almacenar colecciones ordenadas y diccionarios para almacenar pares clave-valor.

## Idea central

Las listas y los diccionarios permiten manejar multiples datos en una sola variable. Las listas son secuencias ordenadas de elementos; los diccionarios asocian claves unicas con valores.
