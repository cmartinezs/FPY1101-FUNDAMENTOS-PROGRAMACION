# Practica

## Ejercicios guiados

1. Crear una funcion sin parametros ni retorno que muestre "Hola mundo".
2. Crear una funcion sin parametros que retorne un numero aleatorio.
3. Crear una funcion que reciba un numero y muestre su cuadrado.
4. Crear una funcion que reciba dos numeros y retorne el mayor.
5. Crear una funcion que calcule el area de un triangulo.

## Ejercicio integrador

Crear un programa con las siguientes funciones:

1. `mostrar_menu()` - Sin parametros ni retorno, muestra las opciones.
2. `obtener_opcion()` - Sin parametros, retorna la opcion elegida (1-4).
3. `calcular_area_cuadrado(lado)` - Con parametros, retorna el area.
4. `calcular_area_circulo(radio)` - Con parametros, retorna el area.
5. `calcular_area_triangulo(base, altura)` - Con parametros, retorna el area.
6. `mostrar_resultado(figura, area)` - Con parametros, sin retorno, muestra el resultado.

El programa principal debe:
- Mostrar el menu
- Pedir una opcion
- Segun la opcion, pedir los datos necesarios
- Calcular y mostrar el resultado

Opciones:
1. Area cuadrado
2. Area circulo
3. Area triangulo
4. Salir
