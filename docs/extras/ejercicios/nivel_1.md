# Nivel 1: Operadores Básicos

Ejercicios que utilizan solo operaciones aritméticas básicas (sin condicionales ni ciclos).

---

### 1. Calculadora de Área de Triángulo
Una empresa de construcción necesita un programa para calcular el área de triangulares de diferentes tamaños. El área de un triángulo se calcula como: `(base * altura) / 2`. Solicita al usuario la base y la altura de un triángulo, y muestra el área calculada.

**Entrada:** Base y altura del triángulo (números decimales)  
**Salida:** Área del triángulo

---

### 2. Conversor de Temperatura
Una estación meteorológica necesita convertir temperaturas de Celsius a Fahrenheit para un informe. La fórmula es: `F = C * 9/5 + 32`. Solicita una temperatura en Celsius y muestra su equivalente en Fahrenheit.

**Entrada:** Temperatura en Celsius  
**Salida:** Temperatura en Fahrenheit

---

### 3. Calculadora de Área de Círculo
Un arquitecto necesita calcular el área de círculos para diseñar elementos circulares. El área es: `π * radio²`. Solicita el radio y muestra el área (use π = 3.14159).

**Entrada:** Radio del círculo  
**Salida:** Área del círculo

---

### 4. Calculadora de Velocidad
Un ingeniero de tráfico necesita calcular la velocidad media de un vehículo. La fórmula es: `velocidad = distancia / tiempo`. Solicita la distancia en km y el tiempo en horas, muestra la velocidad en km/h.

**Entrada:** Distancia y tiempo  
**Salida:** Velocidad media

---

### 5. Conversor de Metros a Pies
Un proyecto de construcción requiere convertir medidas de metros a pies. La equivalencia es: 1 metro = 3.28084 pies. Solicita una medida en metros y muestra su equivalente en pies.

**Entrada:** Medida en metros  
**Salida:** Medida en pies

---

### 6. Calculadora de Área de Rectángulo
Un diseñador necesita el área de rectángulos. El área es: `largo * ancho`. Solicita largo y ancho, muestra el área.

**Entrada:** Largo y ancho  
**Salida:** Área del rectángulo

---

### 7. Calculadora de Perímetro de Cuadrado
Un estudiante necesita calcular el perímetro de un cuadrado para un proyecto escolar. El perímetro es: `4 * lado`. Solicita el lado y muestra el perímetro.

**Entrada:** Lado del cuadrado  
**Salida:** Perímetro

---

### 8. Conversor de Grados a Radianes
Un problema de física requiere convertir grados a radianes. La fórmula es: `radianes = grados * π / 180`. Solicita grados y muestra radianes.

**Entrada:** Grados  
**Salida:** Radianes

---

### 9. Calculadora de Consumo de Combustible
Un conductor quiere calcular el consumo de combustible de su vehículo. El consumo es: `distancia / combustible_usado`. Solicita distancia recorrida en km y combustible usado en litros, muestra el consumo (km/L).

**Entrada:** Distancia y combustible  
**Salida:** Consumo por litro

---

### 10. Calculadora de Volumen de Esfera
Un laboratorio necesita calcular el volumen de esferas para experimentos. El volumen es: `(4/3) * π * radio³`. Solicita el radio y muestra el volumen.

**Entrada:** Radio  
**Salida:** Volumen de la esfera

---

### 11. Calculadora de Precio con IVA
Una factura debe incluir el IVA (19%). Solicita el precio sin IVA y muestra el precio con IVA incluido.

**Entrada:** Precio sin IVA  
**Salida:** Precio con IVA

### 12. Calculadora de Distancia Euclidiana
Un sistema de navegación calcula la distancia entre dos puntos en un plano. La fórmula es: `√((x2-x1)² + (y2-y1)²)`. Solicita las coordenadas de dos puntos y calcula la distancia.

**Entrada:** Coordenadas x1, y1, x2, y2  
**Salida:** Distancia entre puntos

### 13. Conversor de Dólares a Euros
Una casa de cambio necesita un conversor de monedas. La tasa es: 1 dólar = 0.92 euros. Solicita cantidad en dólares y muestra en euros.

**Entrada:** Cantidad en dólares  
**Salida:** Cantidad en euros

### 14. Calculadora de Área de Trapecio
Un ingeniero necesita el área de un trapecio. El área es: `((base1 + base2) / 2) * altura`. Solicita las bases y la altura, calcula el área.

**Entrada:** Base mayor, base menor, altura  
**Salida:** Área del trapecio

### 15. Conversor de Libras a Kilogramos

### 16. Calculadora de Cuota de Préstamo
Un banco calcula la cuota mensual de un préstamo simple: `cuota = capital / meses + (capital * tasa / 100)`. Solicita capital, tasa anual y número de meses, calcula la cuota mensual.

**Entrada:** Capital, tasa anual, meses  
**Salida:** Cuota mensual

### 17. Calculadora de Temperatura con Sensación Térmica
El viento hace que la temperatura se sienta diferente. La sensación térmica aproximada es: `T - (10 * velocidad / 50)` para temperaturas positivas. Solicita T y velocidad del viento, calcula la sensación térmica.

**Entrada:** Temperatura y velocidad del viento  
**Salida:** Sensación térmica

---

### 18. Calculadora de Área de Hexágono Regular
Un diseñador gráfico necesita el área de un hexágono regular. El área es: `(3 * √3 * lado²) / 2`. Solicita el lado y calcula el área (use √3 ≈ 1.732).

**Entrada:** Lado del hexágono  
**Salida:** Área del hexágono

### 19. Calculadora de Hipotenusa

### 20. Calculadora de Volumen de Cilindro
