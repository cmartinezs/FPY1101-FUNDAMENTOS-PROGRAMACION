# Practica

## Ejercicios guiados

1. Pedir el nombre del usuario y mostrar un saludo que diga "Hola [nombre], bienvenido".
2. Pedir un numero y mostrar su triple.
3. Pedir dos numeros y mostrar el promedio de ambos.
4. Pedir el lado de un cuadrado y mostrar su area.
5. Pedir una edad y mostrar si la persona es mayor de edad (>= 18) o menor.

## Ejercicios autonomos

1. Pedir el nombre y el apellido de una persona y mostrar ambos en una sola linea.
2. Pedir tres numeros y mostrar su suma.
3. Pedir el precio de un producto y mostrar el precio con un descuento del 10%.
4. Pedir una temperatura en Fahrenheit y convertirla a Celsius.
5. Pedir el radio de un circulo y mostrar su perimetro (2 × π × radio).
6. Pedir un numero y mostrar su valor absoluto (si es negativo, convertirlo a positivo).
7. Pedir la base y la altura de un rectangulo y mostrar su area.
8. Pedir un numero y mostrar si es divisible por 5.
9. Pedir dos numeros y mostrar cual es menor.
10. Pedir una nota de 0 a 100 y mostrar la letra correspondiente (A=90-100, B=80-89, C=70-79, D=60-69, F=0-59).

## Ejercicio integrador

Crear un programa que:

1. Pida el nombre de un estudiante.
2. Pida tres notas (numeros decimales).
3. Calcule el promedio de las tres notas.
4. Muestre el nombre y el promedio con un mensaje apropiado:
   - Si el promedio es >= 70: "Aprobado"
   - Si el promedio es < 70: "Reprobado"

Ejemplo de salida esperada:

```
Nombre: Juan
Nota 1: 85
Nota 2: 90
Nota 3: 70
Juan tu promedio es 81.67 y estas Aprobado
```
