# Leccion 06 - Funciones

Esta leccion cover funciones en Python: definicion, parametros y retorno. Se presentan los 4 tipos de funciones segun sus parametros y retorno.

## Idea central

Las funciones son bloques de codigo reutilizables que realizan una tarea especifica. Permiten organizar el codigo, evitar repeticiones y hacerlo mas legible.
