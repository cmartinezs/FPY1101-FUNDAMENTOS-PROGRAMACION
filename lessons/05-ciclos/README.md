# Leccion 05 - Ciclos

Esta leccion covers ciclos en Python: `while`, `for` con `range()`, validaciones con ciclos y menus ciclicos.

## Idea central

Los ciclos permiten repetir codigo multiple veces. Python ofrece dos tipos: `while` (repite mientras una condicion sea verdadera) y `for` (repite sobre una secuencia).
