# Leccion 03 - Tipos de datos y conversion

Esta leccion explica los tipos de datos basicos en Python y las conversiones entre ellos.

## Idea central

Python infiere el tipo de dato automaticamente, pero es fundamental conocer los tipos disponibles para manipular datos correctamente y evitar errores.
